# T002 Manuscript Package

This folder contains the working manuscript package for T002.

Files:

- `manuscript.md`: main paper draft.
- `manuscript.tex`: LaTeX manuscript draft.
- `supplementary_pathways_and_experiments.md`: proposed C3 mechanisms, pathway-specific evidence, and experimental plan.
- `references_to_retrieve.md`: papers, DOI links, and local source notes.
- `figures/`: SVG figures generated from synthetic planning data.
- `data/simulated_performance.json`: synthetic data behind the mock graphs.

Important: the figures and performance values are simulated planning data. They are useful for writing, mechanism design, and planning experiments, but they must be replaced by real H-cell, flow-cell, GC, NMR, and characterization data before submission to a journal.

Figure 1 has an Illustrator/PDF-compatible source:

- `figures/Mafa Scheme CuAgRu.ai`
- `figures/fig1_catalyst_concept.pdf`

The LaTeX file uses `fig1_catalyst_concept.pdf` for Figure 1. Figures 2-5 are still SVG planning figures; convert them to PDF or PNG before final LaTeX compilation if you want the rendered graphs inside the PDF.
