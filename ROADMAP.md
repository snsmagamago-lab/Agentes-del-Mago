# ROADMAP

## Phase 1 - Minimal working environment

- Create repository structure.
- Add `AGENTS.md`.
- Add task, handoff, and review templates.
- Add current state files.
- Add first bootstrap task.

## Phase 2 - Local automation

- Add scripts to create tasks.
- Add scripts to prepare Codex and Claude inboxes.
- Add validation of status files.
- Add close-task workflow.

## Phase 3 - Dashboard

- Add local web UI.
- Show tasks, status, handoffs, inboxes, Git state, and logs.
- Allow safe manual actions.

## Phase 4 - GitHub integration

- Add GitHub Actions.
- Run tests on push and PR.
- Keep PR merge under human control.

## Phase 5 - Smarter relay

- Detect handoff changes automatically.
- Suggest next action based on status.
- Add PR helper scripts after credential flow is confirmed.
